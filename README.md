A simple ping pong game made by me. This project was created to improve my programming skills and to track my progress over time.

HOW TO RUN GAME:
To run this game, you need python and pygame installed

1. Install pygame using 'pip install pygame' on the terminal
2. Navigate to the folder
3. Then run the game

Controls:
Left paddle - W (up), S (down)
Right paddle - Up arrow (up), down arrow (down)
R - Reset the game
